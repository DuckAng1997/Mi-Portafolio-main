# Portafolio de Angel Tello

Portafolio profesional estatico enfocado en experiencia real, stack tecnico, trayectoria y proyectos destacados.

## Despliegue en GitHub Pages

1. Sube este proyecto a un repositorio en GitHub.
2. Usa `main` como rama principal.
3. En GitHub ve a `Settings > Pages`.
4. En `Build and deployment`, selecciona `GitHub Actions`.
5. Haz push a `main` y el workflow publicara el sitio automaticamente.

## Archivos principales

- `index.html`: estructura del portafolio.
- `css/estilo.css`: estilos, responsive y tema claro/oscuro.
- `js/main.js`: idioma, animaciones, contadores y navegacion.
- `img/AT.pdf`: CV descargable.
